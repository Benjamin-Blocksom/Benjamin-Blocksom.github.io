The original campaign message was in Brazilian Portuguese, which I translated and then gave this formatting. CSS can be thorny for these as styles are inlined, and HTML is almost exclusively table-based. The trick here is to "code like it's 1999" while keeping an eye on the concerns of the 2020s, like accessibility and responsiveness.


Loudoun Brewery

This holiday greeting was an interesting challenge since I had to create all the copy and images. The holiday message and CTA's were given in the text, with imagery lending a supporting role. Inspiration came from MailChimp's own newsletter and we also used them to run the campaign. The campaign ran on MailChimp and had a 93.5% delivery rate with 45% opens.
